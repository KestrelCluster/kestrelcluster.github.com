KestrelHPC's Personal Package Archives

Ubuntu 10.04, 10.10 and 11.04

  1. Add repository 
    sudo add-apt-repository ppa:kestrel/kestrelhpc
  2. Update packages 
    sudo apt-get update
  3. Install KestrelHPC 
    sudo apt-get install --no-install-recommends kestrelhpc kestrelhpc-ganglia kestrelhpc-openmpi
    Please upgrade frecuently to get the latest fixes.

Debian Squeeze

  1. Add repository 
    echo "deb http://ppa.launchpad.net/kestrel/kestrelhpc/ubuntu maverick main" \ 
> /etc/apt/sources.list.d/kestrelhpc.list
  2. Add PPA's Signing key 
    apt-key adv --keyserver keyserver.ubuntu.com --recv-keys 8475B36D
  3. Update packages 
    sudo apt-get update
  4. Install KestrelHPC 
    sudo apt-get install --no-install-recommends kestrelhpc kestrelhpc-ganglia kestrelhpc-openmpi
    Please upgrade frecuently to get the latest fixes.
