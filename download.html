Installable Packages

KestrelCluster 3.0 Alpha 1 for Debian Squeeze

  1. Add repository 
    $ sudo sh -c 'echo "deb http://ppa.launchpad.net/kestrel/kestrel-3.0/ubuntu oneiric main" \
  > /etc/apt/sources.list.d/kestrelhpc.list'
  2. Add PPA's Signing key 
    $ apt-key adv --keyserver keyserver.ubuntu.com --recv-keys 8475B36D
  3. Update packages 
    $ sudo apt-get update
  4. Install KestrelCluster 
    $ sudo apt-get install --no-install-recommends \
kestrelcluster kestrelcluster-openmpi kestrelcluster-ganglia kestrelcluster-openmpi kestrelcluster-plymouth
    Please upgrade frecuently to get the latest fixes.

Ubuntu

Version 3.0 is not currently available for ubuntu. We are having some troubles with Upstart (Ubuntu's init daemon) and unionfs.
