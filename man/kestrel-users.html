kestrel-users

NAME

KestrelHPC - manual page for KestrelHPC 1.9.13  

SYNOPSIS

kestrel-users options  

DESCRIPTION

OPTIONS:
--list
List Cluster's users
--add user
Adds a new user to the system and to the Cluster
--add-to-cluster user
Adds an existing user to the Cluster
--delete user
Deletes a user
 

SEE ALSO

The full documentation for KestrelHPC is maintained as a Texinfo manual. If the info and KestrelHPC programs are properly installed at your site, the command
info KestrelHPC

should give you access to the complete manual.
