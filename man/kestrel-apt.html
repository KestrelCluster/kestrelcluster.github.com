kestrel-apt

NAME

KestrelHPC - manual page for KestrelHPC 1.9.13  

SYNOPSIS

kestrel-apt options  

DESCRIPTION

OPTIONS:
--install <packages...>
Install packages on default image
--remove
<packages...>
Removes packages from default image
--upgrade
Upgrades packages from default image
--history
Shows a log of the installed and deleted packages
--image <image>
Applies previous commands to <image>
Example :
kestrel-apt --install vim --image "image2" --delete vim --install emacs
Installs vim on the default image. And deletes vim and installs emacs on "image2".
